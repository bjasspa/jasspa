NAME

    $progname - Program file name

SYNOPSIS

    $progname string

DESCRIPTION

    $progname contains the absolute file and path to the MicroEmacs '04 binary file currently being executed. This value is typically used by macros to spawning another MicroEmacs '04 sessions and to determine where MicroEmacs '04 is being executed from.

EXAMPLE

    The following example is used to spawn of another MicroEmacs '04 command to create a C-tags file:- 

      shell-command &cat $progname " \"@ctags\" *.c *.h"
