NAME

    save-all - Save all modified files (with query)

SYNOPSIS

    n save-all

DESCRIPTION

    save-all cycles through all buffers, dictionaries, registry files and current session writing back any changes made. For each buffer, dictionary, session or registry file which has been modified the user is prompted before the changes are saved, a value of y initiates the save, n skips the save.

    The argument n can be used to change the default behavior of save-all described above, n is a bit based flag where:-

    0x01

      Enables the user prompt before the file is saved (default). If this flag is not supplied then all modified files will automatically be written.

NOTES

    save-all is a macro defined in me.emf, using commands save-some-buffers(2), save-dictionary(2), save-session(3), and save-registry(2).
