NAME

    about - Information About MicroEmacs '06

SYNOPSIS

    about

DESCRIPTION

    about displays information about the current MicroEmacs '06 editing session and includes the following information:-

    • Version number and date (YYYY/MM/DD) information for MicroEmacs '06.
    • Global status information including the number of active buffers and global mode status information.
    • Current buffer status information; buffer modes and buffer size information.

    The following is an example output from about.
