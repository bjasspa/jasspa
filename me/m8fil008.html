NAME

    esf - MicroEmacs session file

DESCRIPTION

    MicroEmacs '06 session files are ASCII text files that retain all the buffer context for a previous editing session.

    Every user has their own personal session file in their personal MicroEmacs directory. The session file is located from the MicroEmacs '06 search path defined by $user-path(5), and is named by the variable $user-name(5).
