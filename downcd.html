jasspa-mecd-exec-20060909.zip

A ISO CDROM image containing executable versions of MicroEmacs and NanoEmacs that may be run directly from CD. This is a subset of the full image, source and macro archives (tar.gz) are included with CD-labels. This disc is an alternative to the full images above that supports all pre-built executables with a much smaller download size (excludes spelling, documentation, packages, icons). The disc contains enough information to perform a binary install on all supported platforms in addition to a source build. Download suitable for Dial-up.

Hybrid ISO9660 Disc, UNIX Rock Ridge, Microsoft Joliet, Apple HFS. Where the disc is to be used with HP-UX then the jasspa-mecd-lexe-20060909.zip image should be used.

jasspa-mecd-lexe-20060909.zip

A ISO CDROM image containing executable versions of MicroEmacs and NanoEmacs that may be run directly from CD. The disc is identical to jasspa-mecd-exec-20060909.zip in content, but is constructed with Untranslated filenames to allow direct mounting of the CD-ROM under HP-UX. The disc does invalidate the ISO9660 standards and will not work with MS-DOS because of the long names, the disc is still valid with other UNIX and Microsoft Windows operating systems.

Hybrid non-standard ISO9660 Disc, UNIX Rock Ridge, Microsoft Joliet, Apple HFS with Untranslated filenames. This disc is more suitable for HP-UX systems and may be mounted directly without Rock Ridge extensions under HP-UX.
