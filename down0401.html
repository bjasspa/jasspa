JASSPA's MicroEmacs '04
Release Candidate 1 (20040108-rc1)

A pre-release version of JASSPA's MicroEmacs '04 which may be used for evaluation purposes. Problems with this release candidate should be reported as soon as possible, via the Mailing Lists or directly to support, so that they may be rectified before the final release.

New installations should use the 2002-12 release and then patch to 2004, the files contained here provide sufficient content to patch an existing installation to the new release. A full package release shall be made towards the end of this month.

A limited set of binaries have been provided for convenience, most platforms should be built from source. All components are required.

Refer to What's new for an outline of some new features and changes that existing users should be aware of. For a more complete change summary refer to the History.

Release Candidate 2 (20040109-rc2)

An updated macro release to be used which includes the following. This is a macro change only and should be used instead of the RC1 macros.

  • Fold fix
  • Additional language templates for php and lua
  • Toolbar fixes
  • Addition of string-rectangle command.
