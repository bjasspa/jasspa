Release Candidate 4 (20040206)

    Release Candidate 4. There have been some significant changes in this release with respect to the user set-up. This makes it much easier for new users to set up JASSPAs MicroEmacs correctly. Existing users should review their $MEPATH and $MENAME settings, most platforms will now run without the need to set these environment variables.

    The packaging has changed slightly, the archive metree contains a standard MicroEmacs macro tree, this contains the macros and help files and may be directly unpacked in /usr/local (UNIX), c:\ (DOS) or C:\Program Files\ (Windows) for the JASSPA directory.

    Minimum download is:

    • installation package
    • OR metree and executable image
    • OR metree and mesrc to build your own image.

    Please read the Installation Notes for further information
