When downloading the distribution then take both the source and the macro components, these should NOT be mixed with a previous release.

Files supplied in the UNIX tar.gz format may be downloaded to Microsoft Windows/DOS environments, use Winzip or native UNIX tools to unpack the files.

Download Problems ??
There appear to be some download (CR/CRLF) problems from Microsoft Windows platforms using Netscape (4.7 ??). If you encounter problems then try downloading using IE5, use an ftp utility (i.e. CuteFTP, or AbsoluteFTP) or move onto a UNIX box. If you encounter corrupted downloads then make sure that the file size matches the size specified on the server (byte size NOT block size as reported by some Operating Systems).

Linux
Linux users should note the following:-

  • RedHat Linux RPM and SRPM packages available from http://www.hardline.org/
  • Mandrake 8.0 the binary supplied crashes on execution. It appears that there is an incompatibility in the binary image, this may affect other new 2.4.x Linux distributions, download the source code and build executable image for your distribution (./build in the source directory).
