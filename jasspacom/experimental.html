This page contains Beta versions of the JASSPA MicroEmacs software that have not reached a golden status that is sufficiently mature to form part of a full release. The content is made available as early access for testing. The content presented should stable enough for general use and beta testing. Unless you are familiar with JASSPA's MicroEmacs then it it is inadvisable to download from this page, the latest release is preferred.

When downloading the distribution then take both the source and the macro components, these should NOT be mixed with a previous release. Installed versions of the 12/2002 release may be patched with these files.

Files supplied in the UNIX tar.gz (.tgz) format may be downloaded to Microsoft Windows/DOS environments, use Winzip or native UNIX tools to unpack the files.

Building from source
If your system does not have a binary image then you may build one from the source. For UNIX a simple ./build from the source directory will build an executable image for a supported target.

Download Problems ?? There appear to be some download (CR/CRLF) problems from Windows platforms using Netscape (4.7 ??) with WinZip. If the archive appears to be corrupt then check the downloaded file byte size with that stated on the site (byte size NOT block size as reported by some Operating Systems). If you encounter problems then try some of the following:-

  • Disabling Smart CR/LF in WinZIP.
  • Download using another browser i.e. Microsoft IE5 or Opera.
  • Download from UNIX
