News - 25th January 2000

More Fonts ...

    We have just started prototyping support for multiple fonts within the rendered window, including:

    • bold
    • italic
    • underline

    This set will probably be extended to include reverse and light corresponding to the standard UNIX Termcap characters.

    An initial investigation into the feasibility of embedding this into the current release has been perform on UNIX and MS-Windows platforms, as shown in the following screen shots:-


    Unix, running through Exceed to Windows desktop


    Microsoft Windows

    I'll be the first to admit we have to work on the color schemes, but you get the general idea !! (Sorry, Sherwood Forest is my current favorite)

    Hopefully this will drop out towards the end of February or beginning of March once we have sorted out the mechanics and ironed out all of the bugs -- and of course put the documentation in place.

LaTeX Documentation

    Work has just started on pulling together all of the LaTeX documentation in an attempt to catch up with all of the new features that have recently been added to the hklatex.emf template. (Thanks to Detlef for this). This should be complete in a couple of weeks. After an initial scout around I was surprised to find just how far this has progressed.

Character Set Support

    Work has been progressing with multiple character set support, mainly for foreign languages. This has meant some internal reorganization of the source code in order to accommodate the changes. This work is just about complete and is appearing in the most recent beta versions.

ftp/http File Loading

    In the last few weeks, the finishing touches of the ftp support have been added. Driven by the requirement to be able to hack files on UNIX boxes from Windows machines not equipped with X-Servers (or UNIX boxes lacking Samba support). This later manifested itself in being able to move files with ftp from the local to remote file systems by the simple addition of a few macros.

    This work will appear in the next beta release (approx 25th Jan).
