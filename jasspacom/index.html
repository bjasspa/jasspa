Jasspa's MicroEmacs

Welcome to www.jasspa.com, the web site for the distribution of JASSPA's MicroEmacs.

This site first appeared on-line on 12th January 2000.

Jasspa's MicroEmacs is an Emacs editor biased towards UNIX users, working across platforms by providing a consistent interface under UNIX, Microsoft Windows '95/'98/NT and DOS operating systems.
