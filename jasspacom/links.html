Unix tools

    For DOS users the UNIX tools (diff/grep etc.) may be obtained from:-

    • ftp://ftp.cdrom.com/pub/garbo/garbo_pc/unix/uxutl23a.zip (238 Kb)
    • ftp://ftp.cdrom.com/pub/garbo/garbo_pc/unix/uxutl23b.zip (227 Kb)
    • ftp://ftp.cdrom.com/pub/garbo/garbo_pc/unix/uxutl23c.zip (221 Kb)
    • ftp://ftp.cdrom.com/pub/garbo/garbo_pc/unix/uxutl23d.zip (160 Kb)

    One awk-port; the Gnuish project has 16 bit and 32 bit versions of gawk(1) in:

    • ftp://mirrors.aol.com/pub/simtelnet/gnu/gnuish/gawk303x.zip (1997, 495K)

    For Windows tools (grep/diff etc.) may be obtained from:-

    • http://unxutils.sourceforge.net/

    The cygwin environment is probably the best environment for UNIX shells and tools.

    • http://www.cygwin.com

Compiler tools

  • A Windows C Compiler is now free for download for private use. The Borland 5.5 C-Compiler may be obtained from http://www.inprise.com. All JASSPA releases 2000/03/22 and later may be built with this compiler using the makefile win32b55.mak. To build then use the command line "make -f win32b55.mak".
  • Sun Microsystems now offer Sun Studio for free. This is the best compiler for Sun Solaris.
