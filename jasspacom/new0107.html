What's New

    News - 16th July 2001

    New WWW Pages

      It's been over 1 year since our last release. With the new release we have also updated our WWW pages to make them a little easier to navigate and provide some more content.

      Apologies for the long delay - too busy with other work to provide the amount of time that we would like to.

    Simplified Macro Templates

      The latest release includes a new set of macro templates for the programming languages. This simplies the chore of defining common key bindings, menu items, folding and automated comment reformatting. The template files themselves are now consideravbly smaller and should be a little easier for users to define their own templates with a common set of functionality.

    CVS

      Support for CVS has now been incorporated into MicroEmacs through the new cvs macro. This provides a front end to CVS via the file browser (F10) and via the command line.

      The graphical diff utility (gdiff) may be used to difference the archive and currently checked out file.

    Key Stroke Rebinding

      Following some requests from users for CUA bindings (i.e. Microsoft Windows bindings) then we have added support for a complete unbinding and rebinding of all of the MicroEmacs key strokes. As an example a set of nedit bindings have been provided which are close to standard Windows bindings. It should now be possible for users to create a set of Windows binding for the user based on nedit. Sorry we have not done this for you - our preference are standard Emacs bindings !!
