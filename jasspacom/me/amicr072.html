QUESTION (31)

    Microsoft Drag and Drop; is it supported ??

ANSWER

    MicroEmacs '04 supports Microsoft drag and drop interaction. Multiple files and directories may be dragged from Microsoft Explorer (or other application) and dropped into a buffer window. The destination buffer window is the window in which the dropped file(s) are displayed.

    Note if the user is currently on the command line, then the command line operation is aborted in order to facilitate the dropped files.
