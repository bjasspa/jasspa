QUESTION (36)

    Non-English Languages; What font should I select ??

ANSWER

    MicroEmacs '04 has only been tested with Western Lanuguages only. Within the Microsoft Windows environment an ANSI type font should be selected, assuming of course that the characters required are in the ISO-Latin character set. UNIX typically supports ISO-Latin character sets.
