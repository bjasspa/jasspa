NAME

    string-rectangle - Add a string vertically.

SYNOPSIS

    string-rectangle

DESCRIPTION

    string-rectangle inserts a string at the same column position on each line of a selected region. The command operates on a selected region and prompts the user for a string to insert at the column position at the start of the region. The user defined string is then inserted that the same column position for all lines that exist in the region. This is generally used to pre-pend lines with new text.

    The insertion column is the column at the start of the region with the lowest line number. (i.e. uppermost line of region).
