NAME

    .calc.result - Last calc calculation result

SYNOPSIS

    .calc.result integer

DESCRIPTION

    .calc.result is used to store the result of the last calculation made by calc(3).

    The "LR" (Last Result) in the next calculation is substituted with this value.
