NAME

    $global-scheme - Default global buffer color scheme.

SYNOPSIS

    $global-scheme schemeNum; Default is 0

DESCRIPTION

    $global-scheme defines the default buffer color scheme to schemeNum, a color scheme defined by add-color-scheme(2).
