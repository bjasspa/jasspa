NAME

    narrow - Buffer contains a narrow

SYNOPSIS

    narrow Mode

    N - mode line letter.

DESCRIPTION

    This mode can not be set and is used to indicate whether the buffer contains a narrow, created by the narrow-buffer(2) command.
