NAME

    $mode-chars - Mode characters

SYNOPSIS

    $mode-chars

DESCRIPTION

    $mode-chars is a read-only variable that returns a string that contains all of the buffer mode characters. The variable is typically used in the macros to determine the integer value of a buffer mode, the string is defined as follows: 

      AaBbcYzdDeEfHIJkMnNOPQrSTtU12345678VW

    The index into the string matches $show-modes(5).
