NAME

    edf - MicroEmacs spelling dictionary file

SYNOPSIS

    lsdmenus.edf
    user.edf

DESCRIPTION

    The spelling dictionary files are given the extension .edf. These are binary files read by MicroEmacs '06 and cannot be edited directly.

    MicroEmacs '06 is supplied with a dictionaries for various languages. It is recommended that these dictionaries are not modified, a personal dictionary is used and modified instead.

    A personal dictionary, user.edf, is automatically created in the users directory for additional spelling information.

FILES

    The standard dictionary files lsdm<language><country>.edf are located in the MicroEmacs '06 home directory.

    User's may create their own dictionary files by shadowing the home directory file with their local dictionary(s) located in a personal MicroEmacs '06 directory. See $search-path(5).
