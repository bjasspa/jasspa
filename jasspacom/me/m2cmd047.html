NAME

    execute-named-command - Execute a named command

SYNOPSIS

    n execute-named-command "command-string" esc x

DESCRIPTION

    execute-named-command command prompts the user for the name of a command to execute and then executes the command n times. MicroEmacs '05 offers command completion and history facilities, see ml-bind-key(2).
