NAME

    $show-modes - Select buffer modes to display

SYNOPSIS

    $show-modes "bit-string"; Default is ""

DESCRIPTION

    $show-modes defines which buffer modes are displayed on the mode-line. The index into the string matches $mode-chars(5).

SEE ALSO

    $user-setup(3), $mode-chars(5), $mode-line(5).
