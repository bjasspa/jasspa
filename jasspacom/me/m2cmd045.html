NAME

    execute-file - Execute script lines from a file

SYNOPSIS

    n execute-file "file" (esc /) 

DESCRIPTION

    execute-file executes script lines from the given file n times in succession, this is the normal way to execute a MicroEmacs '04 script. The command prompts for a file name, and will then search for <file>[.emf] in the search path. If the file is found then the file is loaded and the buffer is executed n times.
