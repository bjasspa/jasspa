GAMES

    The following is a list of all of the games provided by MicroEmacs:

    Mahjongg(3) MicroEmacs '06 version of the solitaire Mah Jongg game
    Match-It(3) MicroEmacs '06 version of the Match-It game
    Metris(3) MicroEmacs '06 version of the falling blocks game
    Patience(3) MicroEmacs '06 version of Patience (or Solitaire)
    Triangle(3) MicroEmacs '06 version of Triangle patience game
