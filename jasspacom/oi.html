OpenIndiana Small System Server Build

The following document contains some notes to configure OpenIndiana on a HP N40L MicroServer. The server provides DNS, E-mail, web services, calendar, address book, file storage and print services to mobile devices such as iPad and Android.

These notes are made available on an as-is basis with no warranty in the hope that they may be useful to somebody else building their own server. They are based on notes accumulated as I built my own server, this turned out to be time consuming as there was no quick reference in one place and required a lot of searching to find information.

  • oi_setup.pdf OpenIndiana Small System Server Build (PDF 1.7MB)
If you find any errata, or have anything else to add, then please drop me an E-Mail ("jon" at this site address) and I will endeavour to keep the document up to date.
