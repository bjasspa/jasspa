When downloading the distribution then take both the source and the macro components, these should NOT be mixed with a previous release. Users upgrading from 2001 (or older) versions must also download the spelling dictionaries as this release is no longer compatible with the older dictionaries.

Files supplied in the UNIX tar.gz format may be downloaded to Microsoft Windows/DOS environments, use Winzip or native UNIX tools to unpack the files.

Building from source

    If your system does not have a binary image then you may build one from the source. For UNIX a simple ./build from the source directory will build an executable image for a supported target. A build.bat and Microsoft Developer project are provided for Microsoft environments.

Upgrading from a previous release

    If you are upgrading from a previous release then some of the binary data in the registry file <user>.erf file may have changed. Delete your existing <user>.erf file and re-configure using M-x user-setup (or via the menu Help->User Setup). Failure to do so will mean that some of the buffer modes are incorrect.

    This notice also applies to the previous Release Candidates for '04 on Microsoft Windows environments where a Wheel Mouse is used. In this case the default key binding is changed and needs to be re-made via M-x user-setup.

    There have been some significant changes in this release with respect to the user set-up. This makes it much easier for new users to set up JASSPAs MicroEmacs correctly. Existing users should review their $MEPATH and $MENAME settings, most platforms will now run without the need to set these environment variables.

Quick Download Guide

    The packaging has changed slightly in the March 2004 release. The archive metree contains a standard MicroEmacs macro tree, comprising the macros and help files and may be directly unpacked in /usr/local (UNIX), c:\ (DOS) or C:\Program Files\ (Windows) for the JASSPA directory.

    Minimum download is:

    • installation package
    • OR metree and executable image
    • OR metree and mesrc to build your own image.

    Please read the Installation Notes for further information
