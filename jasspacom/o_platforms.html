The following table identifies the makefiles that are included in the source distribution and the currently officially supported platforms

Platform Makefile Description

hpux10.gmk (GNU) 		Hewlett Packard [UNIX]

hpux11.gmk (GNU) 		Hewlett Packard [UNIX]

irix5.gmk (GNU) 		Silicon Graphics IRIS [UNIX]
IRIX 6.x irix6.mak (native)

Windows 3.x (win32s) win32sv4.mak Microsoft Developer Studio V4.0
  • Items marked as (native) imply the platforms native make and compiler tools.
  • Items marked as (GNU) imply GNU make and GNU gcc compiler tools.
  • For other UNIX platforms not mentioned above, the UnixWare build apparently provides the best starting point.
  • Microsoft Developer Studio V4.0 should be based on win32v5.mak, the library and include paths within the makefile will require modification.
  • All Microsoft Developer Studio makefiles build with "nmake -f <xxx.mak>". Ensure that the compiler environment is defined in the DOS command line space before attempting a build.
  • All 32-bit Windows versions support long filenames.

Window Mode Support

Window manager support is available on the following platforms:-

    Platform
    Windowing Support
    UNIX Native X-Windows and Terminal modes
    Cygwin Terminal and X-Windows modes
    DOS Console mode only
    Windows '95/'98/NT/2K/XP Window and Console modes
    Windows 3.11 (win32s) Window mode; use DOS for console.
