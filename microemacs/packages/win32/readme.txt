For building the install for windows they you need the following:

Microsoft HTML Compiler
Inno Setup - isetup-5.3.4

Extract  the  jasspa-metree  in the  "JASSPA"  directory  and  move all of the
subdirectories of "jasspa" to MicroEmacs".

Add the following files and change to Windows line ending:

COPYING
build.txt
cygwin.txt
faq.txt
infolist.txt
jasspa-mesrc-20090909.zip
jasspame.pdf
license.txt
me.chm
me.hlp
me32.exe
patch.txt
readme.txt

The simple build the bundle using Inno.
